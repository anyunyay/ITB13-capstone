# ITB13-capstone

## Samahang Magugulay at Magmamais ng Cabuyao E-Commerce and Logistics Platform
This web-based platform is to support the farmers of Samahang Magugulay at Magmamais ng Cabuyao. It allows them to sell their products directly to consumers, ensuring fair pricing and reducing the need for middlemen. The system improves transaction processes, increases efficiency, and strengthens the cooperative’s presence in the digital economy, helping improve the members’ livelihoods.
